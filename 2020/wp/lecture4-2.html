최창락

최창락(1999.99.99)은 대전에서 태어난 대한민국의 남자다.

목차

  • 생애
  • 게임
  • 수업

생애

최창락은 1999년 99월 99일에 대전의 변두리 지역에서 앵앵 울면서 태어났다. 아버지와 어머니가 있고, 위로 누나가 있었다. 학교는 대전에서 18년간 다녔다.

이번학기 수업진행 진도

개발한 게임

  • 가약스
  • 나비스온라인
  • 리니지2레볼루션
  • 킹오파 올스타즈

수업

데이터 싸이언스를 위한 파이썬

파이썬을 가지고 데이터를 읽어오고 가공하여 결과를 출력하는 과정을 배우는 수업

웹 프로그래밍

HTML5, CSS3, Java Script를 이용하여 웹페이지를 표출하는 방법을 배우는 수업

HTML5는 웹페이지를 구축하는 구조화 담당

CSS3는 웹페이지를 보여주는 보여주기 담당

Java Script는 웹페이지의 결과값을 필요에 따라 바꾸는 역할
